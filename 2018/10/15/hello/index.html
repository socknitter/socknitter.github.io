Hexo in GitHub Pagesで静的サイトを運用してみる。

みちくさ
2018-10-15
Hexo

面白そうだったので、作ってみた。
自前でテンプレートを作ってみたいな、というのが今後の課題。

使用したもの

  • Hexo in GitHub Pages
    • Theme : Cactus Light

Hexoってなんぞや、というと日本語では「静的サイトジェネレーター」と呼ばれていて、Hexoの公式では「ブログフレームワークで、投稿を書くと静的なファイルを数秒で生成するよ」と言っている。Node.jsで作られているそうです。
参考：https://hexo.io/docs/index.html

ちなみに、静的サイトジェネレーターは他にもあって、
Rubyで作られているJekyllや、Pythonで書かれているPelicanなどがある。
